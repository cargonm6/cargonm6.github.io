Algoritmo Dijkstra

Esta página simula el comportamiento de una red, donde cada nodo es un router y cada enlace tiene un coste o peso asociado. Dijkstra (https://es.wikipedia.org/wiki/Algoritmo_de_Dijkstra) permite calcular la ruta óptima entre dos nodos, y sirve de fundamento para todos los algoritmos de enrutamiento de estado-enlace.
